Ghost in the Machine

Make your website haunted for halloween! Scare your friends with a simple javascript function and a bit of CSS black magic.

CAUTION: LOUD AUDIO!

Step 1: Your webpage

Choose which webpage you want to haunt! All you will need to do is drop the following code into your html file.

<div class="ghost_container"> <img class=ghost src="ghost_pic.png" alt="A picture of a scary ghost!"> </div>

This is the haunting image that will appear when your ghost reveals itself. Add a picture of your choosing or you can download our cute ghost, Booful.

Step 2: Your CSS Stylesheet

Now is time to get your ghost in the spirit of haunting. We are going to change the initial display of your ghost to hidden so that it can pop out and surprise your site's visitor.

.ghost_container { width:100%; position:absolute; text-align: center; visibility: hidden; } .ghost { z-index: 10; display: block; width: 550px; position: relative; margin-left: auto; margin-right: auto; }

Step 3: Your Javascript file

This Javascript file will bring our ghost to life with sound and sight! The following code sets a timer for 5 seconds using setTimeout that will trigger the appearance of your ghost and the ear piercing scream when the visitor goes to your site! Download our ear piecing scream or use your own sound!

var ghost = document.getElementsByClassName("ghost_container")[0]; var sound = new Audio("file.wav"); //Shows ghost and plays sound after one second setTimeout(function () { sound.play(); ghost.style.visibility = "visible"; console.log('show'); }, 5000); //Hides ghost one second after appears setTimeout(function () { ghost.style.visibility = "hidden"; console.log('hide'); }, 6000);

This way the ghost will make an appearance 5 seconds after your user loads the page and then quickly disappear again into the shadows.

Not scary enough? Try randomizing your javascript timer for even more suspense!

Step 4: Final steps

Make sure you are linking both your CSS and javascript files to your html page in the header!

<head> <link rel="stylesheet" type="text/css" href="main.css"> <script src="main.js"></script> </head>

Happy Haunting!
