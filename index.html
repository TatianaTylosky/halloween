Ghost in the Machine

Make your website haunted for halloween! Scare your friends with a simple javascript function and a bit of CSS black magic.

Step 1: Your webpage

Choose which webpage you want to haunt! All you will need to do is drop the following code into your html file.

<div class=ghost>BOO!</div>

This is the haunting text that will appear when your ghost reveals itself. OPTIONAL: Add a super scary picture of your choosing to add all the more fright and terror!

Step 2: Your CSS Stylesheet

Now is time to get your ghost in the spirit of haunting. We are going to change the initial display of your ghost to hidden so that it can pop out and surprise your site's visitor.

.ghost { font-size:500px; color:white; position: fixed; z-index: 10; display: none; }

Step 3: Your Javascript file

This Javascript file will bring our ghost to life with sound and sight! The following code sets a timer for 10 seconds that will trigger the appearance of your ghost and the ear piercing scream!

setTimeout(function () { var ghost = document.getElementsByClassName("ghost")[0]; var snd = new Audio("file.wav"); snd.play(); ghost.style.visibility = "visible"; }, 3000); setTimeout(function () { var ghost = document.getElementsByClassName("ghost")[0]; ghost.style.visibility = "hidden"; }, 5000);

This way the ghost will make an appearance 10 seconds after your user loads the page and then quickly dissapear again into the shadows.

Not scary enough? Try randomizing your javascript timer for even more suspense!

Step 4: Final steps

Make sure you are linking both your CSS and javascript files to your html page in the header!

<head> <link rel="stylesheet" type="text/css" href="main.css"> <script src="main.js"v></script> </head>

Now go ahead and test out your ghost a few times before you share your goul with a friend!

Give your text an eerie glow by applying some CSS to your font!

.text-glow { text-shadow: 1px 1px 15px white; }
